前置作業

申請帳號

  • 申請github 帳號

本機安裝

  • Node.js 下載安裝LTS版(Long Term Support)
  • git

相關設定

  • 產生自己的SSH Key
  1. 打開git-bash.exe

  2. 輸入以下指令

    • ssh-keygen -t rsa -C "郵箱地址@mail.com"

    • 出現下列訊息按Enter即可

    • 若不需密碼按Enter跳過即可

    1. 可以在c:\users\yourname\ .ssh\找到SSH KEY
      • 私鑰檔名：id_rsa
      • 公鑰檔名：id_rsa.pub
  • 與git連接
  1. 使用文字編輯軟體開啟id_rsa.pub這個檔案，將其內容複製起來
  2. 登入gitgub.com後，右上角圖像 => Settings => SSH and GPG keys => New SSH key 貼上!!
